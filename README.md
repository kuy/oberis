Tetris + [Obelisk.js](https://github.com/nosir/obelisk.js) with Redux.

![Oberis Preview][oberis_img]

## Usage

```
$ npm install
$ npm start
```

Open `http://localhost:8080` in your browser.

## Development

### Unit Testing (mocha + power-assert)

```
$ npm test
```

### Static Type Checking (flow)

```
$ npm run flow
```
